# JSGame

- как используете сохранения

Для работы с сохранениями данных о игроке (хп и очки) создан класс Storage, он имеет методы:
    * save - метод для сохранения данных о игроке
    * load - метод для загрузки данных, возвращает игрока с показателями из сохранения 
             (если нет данных, возвращает нового игрока)
Storage используется в обработчиках кнопок, которые отвечают за загрузку и сохранение

- какие 2 механики создали

1) Основная механика боя
    1.1) нанесение ударов по клику на монстра и периодическое получение урона от монстров
        - орк - сильный урон, 3 
        - ящерица - средний урон, 2 
        - слайм - слабый урон, 1
    1.2) набор очков за побеждённых монстров: 
        - орк - большая награда (300 очков) 
        - ящерица - средняя награда (200 очков) 
        - слайм - маленькая награда (100 очков)
2) Механика восстановления здоровья за счёт покупки зелья за очки
        - когда у игрока достаточно очков (1000 и больше), то зелье подсвечивается
        - 1 зелье стоит 1000 очков
        - 1 зелье соответствует полному лечению

- какие клавиши задействованы и для каких действий

Space - покупка зелья
Esc - вызов контекстного меню в режиме игры
